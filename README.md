# Crocker-Fellowship-iGem
This repository is meant to house the code for our iGem device and give use good version control. Made possible through the Crocker fellowship at BYU. 

## Rules:
- Don't commit/push changes to the main branch
- Preferably refrain from commit/pushing changes directly to dev branch
  - Pull requests (PRs) to the dev branch are the best way to push code
  - You may commit/push to dev for a very small change that you are certain will work flawlessly.
